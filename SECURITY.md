# Security Policy

## Supported Versions

Use this section to tell people about which versions of your project are
currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

Note: Only the latest version is actively supported with security fixes. Please update ComfyUI and this custom node pack regularly. Older versions might work, but are not guaranteed to work.

## Reporting a Vulnerability

We take all security issues seriously. If you have found a potential vulnerability, please report it privately:
Email: business@zanno.se.

We commit to providing an initial response within 48 hours and an estimate for a fix (or a detailed explanation if it is declined) within 7 days.
